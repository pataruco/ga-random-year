## How to install

1. Git clone this repo

   ```sh
   git clone git@github.com:pataruco/ga-random-year.git
   ```

2. Go to the folder

   ```sh
   cd ga-random-year
   ```

3. Install dependecies

   ```sh
   yarn
   ```

## How to run

- To run locally
  ```sh
  yarn dev
  ```
- To build bundle
  ```sh
  yarn build
  ```

## How to deploy

1. Create a git tag starting with a capital **R**

   ```sh
   git tag R1
   ```

2. Push tag

   ```sh
   git push --tags
   ```
